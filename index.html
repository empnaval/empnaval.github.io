HTML table tags

HTML tag Function
<table> Initiates a table
<thead> Contains the table's header
<tbody> Contains the table's main data
<th> Defines column headers
<tr> Creates a new row
<td> Creates a new feild in the row

Simantic HTML tags

These are HTML tags that are used instead of <div> tags. They offer an alternative that is more readable.

HTML tag Use case Example
<header> Used for the header section of the page <header>
<h1>This is the Title of the page</h1>
<h3>This is the sub-heading of the page.</h3>
</header>
<nav> Used for the navigation section of the page <nav>
<ul>
<li>About page</li>
<li>Contace page.</li>
</ul>
</nav>
<article> Used for a self contained/independant section <article>
<h3>This is the heading of the article</h3>
<p>This is the meat of the article.</p>
</article>
<aside> Used for a remark that is tangentially relevant, but not completely indepnedant of the context <p>This is a paragraph of text.</p>
<aside>This is a remark that is relevant to the paragraph above.</aside>
<footer> Used for the footer section of the page <footer>This is the footer of the page, could contain contact details and developer information.</footer>
